# 📘 AI Edu-Hub 프로젝트 통합 매뉴얼 (v1.0)

> **"선생님의 퇴근을 당기고, 학생의 성적을 뒤집는 단 하나의 AI 학습 관리 솔루션"**

---

## 1. 프로젝트 개요
**AI Edu-Hub**는 문제 생성, 관리, 채점, 분석을 아우르는 올인원 SaaS 교육 플랫폼입니다. 교사에게는 업무 경감을, 학생에게는 초개인화 학습 경험을, 학부모에게는 투명한 데이터를 제공하는 것을 목표로 합니다.

- **플랫폼 형태**: 반응형 웹, PWA(Progressive Web App)
- **주요 타겟**: 교사, 학생, 학부모

---

## 2. 시스템 아키텍처 & 기술 스택

### 🏗️ Architecture
프로젝트는 현대적인 웹 기술 스택을 바탕으로 높은 확장성과 안정성을 확보하고 있습니다.

- **Frontend**: **Nuxt.js (Vue 3)**
  - SSR(Server Side Rendering)을 통한 빠른 초기 로딩 및 SEO 최적화.
  - PWA 지원으로 모바일 친화적 환경 제공.
- **Backend**: **NestJS (Node.js)**
  - TypeScript 기반의 모듈화 아키텍처로 엔터프라이즈급 안정성 제공.
  - RESTful API 및 백그라운드 스케줄링 처리.
- **Database & ORM**: **MySQL + Prisma**
  - 관계형 데이터베이스의 안정성과 Prisma의 타입 안정성 결합.
- **AI Engine**: **OpenAI GPT-4o**
  - Vision API를 활용한 문제 이미지 OCR 및 수식 추출.
  - 의미 기반의 주관식 자동 채점 및 해설 생성.

## 3. 핵심 모듈 및 주요 기능

### 👨‍🏫 교사 관리 시스템 (Teacher Workflow)
- **0.5초 디지털 문제 생성**: 문제집 이미지를 촬영/업로드하면 AI가 OCR을 통해 텍스트 및 수식을 즉시 추출하여 DB에 저장합니다.
- **문제 은행 및 편집**: 추출된 문제를 고급 에디터(LaTeX 지원)를 통해 가공하고 유튜브 강의 URL 등의 멀티미디어를 연결할 수 있습니다.
- **고사집/문제집 배포**: 생성된 문제들을 묶어 학급별로 과제나 시험지로 즉시 배포합니다.

### ✍️ 학생 학습 시스템 (Student Workflow)
- **반응형 문제 풀이**: PC와 모바일 모두에 최적화된 인터페이스로 문제를 풀고 제출합니다.
- **AI 스마트 채점**: 주관식 답안의 맥락과 키워드를 분석하여 정 오답을 판별합니다.
- **에빙하우스 복습 스케줄러**: 오답 데이터를 기반으로 1일, 3일, 7일 뒤 자동으로 복습 과제를 생성하여 장기 기억을 돕습니다.

### 📊 학부모 안심 리포트 (Parent Workflow)
- **실시간 학습 현황**: 자녀의 과제 완료 시 즉시 채점 통계 및 약점 분석 데이터를 확인합니다.
- **자동 알림 파이프라인**: 학습 보고서가 생성되면 이메일 또는 알림톡을 통해 자동으로 고지됩니다.

## 4. UI/UX 디자인 시스템 (Premium Experience)

최근 업데이트를 통해 대시보드와 관리 도구에 **프리미엄 디자인 시스템**이 적용되었습니다.

- **Dashboard**: 
  - 실시간 명언 API 연동 및 타이핑 애니메이션 효과.
  - 최근 활동 로그 및 학습 통계의 시각적 시각화.
- **Manager Navigation**: 
  - Glassmorphism(배경 블러) 효과가 적용된 통합 네비게이션.
  - 활성 상태에 따른 다이나믹한 시각적 피드백.
- **Action Button Groups**:
  - '생성' 및 '삭제' 등의 주요 액션을 일관된 디자인의 버튼 그룹으로 통합.
  - Indigo 그라데이션과 부드러운 호버 애니메이션 적용.
- **LaTeX Rendering**: 복잡한 수학 수식과 기호를 깔끔하게 표현하는 전용 렌더링 엔진 탑재.
